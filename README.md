# Anti-Sleep/Hazard System for Drivers

This is the initial commit for an anti-sleep and hazard detection system for drivers.

## Project Overview

This project aims to improve driver safety by detecting drowsiness and potential hazards.

### Current Features
*   This is just the start of the project.

### Planned Features
